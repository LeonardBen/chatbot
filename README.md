## Installation & usage

1. Clone repository

```bash
  git clone https://github.com/LeonardBen/chatbot
  cd chatbot
```

2. Install dependencies
```bash
  npm install
```

3. Start the project
```bash
  npm run dev
```

4. Get all commands
```bash
  help
```
